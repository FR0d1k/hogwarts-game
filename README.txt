Описание игры:
{
Hogwarts: Entrance Exam — это головоломка на угадывание пятизначного кода, оформленная в стиле вселенной Гарри Поттера.
Как играть: На главном экране нажмите "New Game". Появится запечатанный конверт, который затем раскроется — внутри письмо от профессора Макгонагалл. В письме сказано, что вы приняты в Хогвартс, но сначала нужно пройти вступительное испытание: разгадать секретный пятизначный код.
Головоломка: Введите любую комбинацию из пяти цифр (клавиши 0–9, Backspace для удаления) и нажмите Enter. После каждой попытки появятся пять черепов — они подсказывают результат:
Череп с горящими глазами — цифра угадана и стоит на правильном месте
Череп, пробитый стрелой — цифра есть в коде, но стоит не на своём месте
Пустой череп — такой цифры в коде нет вообще
После каждой попытки даётся 6 секунд чтобы запомнить результат, затем поле ввода очищается. Всего дано 5 попыток.
Итог: Угадали код — поздравления, вы зачислены в Хогвартс. Не угадали за 5 попыток — провал, придётся попробовать в следующем году. В обоих случаях можно вернуться в главное меню и начать заново.
}
Процесс разработки
1. Выбор концепции и инструментария
Выбрана тема игры — головоломка на угадывание 5-значного кода в стиле Гарри Поттера. Для реализации использована библиотека Raylib на языке C++.
2. Проектирование структуры
Определён список игровых состояний (GameState): меню, показ конверта, письма, головоломка, победа и поражение. Для хранения данных разработана структура Puzzle.
3. Реализация игровой логики
Реализована генерация случайного кода и функция проверки CheckGuess, возвращающая результат: 2 — цифра на своём месте, 1 — не на том месте, 0 — отсутствует.
4. Разработка интерфейса
Для каждого состояния создан отдельный экран с фоном и шрифтом в стиле Гарри Поттера. Результат попытки отображается через текстуры черепов трёх видов.
5. Звуковое сопровождение
Для каждого состояния подобрана отдельная музыкальная тема с переключением через функцию SwitchMusic.

Возникшие трудности
В процессе разработки была предпринята попытка добавить русскоязычный интерфейс. Настроить корректное отображение русского текста не удалось из-за проблем с кодировкой — символы отображались некорректно. Разобраться с настройкой кодировки в отведённое время не получилось, поэтому интерфейс был оставлен на английском языке.
